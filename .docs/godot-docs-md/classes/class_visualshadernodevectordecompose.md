## VisualShaderNodeVectorDecompose

**Inherits:** [VisualShaderNodeVectorBase](class_visualshadernodevectorbase.md#class_VisualShaderNodeVectorBase) **<** [VisualShaderNode](class_visualshadernode.md#class_VisualShaderNode) **<** [Resource](class_resource.md#class_Resource) **<** [RefCounted](class_refcounted.md#class_RefCounted) **<** [Object](class_object.md#class_Object)

Decomposes a [Vector2](class_vector2.md#class_Vector2), [Vector3](class_vector3.md#class_Vector3) or 4D vector (represented as a [Quaternion](class_quaternion.md#class_Quaternion)) into scalars within the visual shader graph.

### Description

Takes a `vec2`, `vec3` or `vec4` and decomposes it into scalar values that can be used as separate outputs.
