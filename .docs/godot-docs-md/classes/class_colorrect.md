## ColorRect

**Inherits:** [Control](class_control.md#class_Control) **<** [CanvasItem](class_canvasitem.md#class_CanvasItem) **<** [Node](class_node.md#class_Node) **<** [Object](class_object.md#class_Object)

A control that displays a solid color rectangle.

### Description

Displays a rectangle filled with a solid [color](class_colorrect.md#class_ColorRect_property_color). If you need to display the border alone, consider using a [Panel](class_panel.md#class_Panel) instead.
