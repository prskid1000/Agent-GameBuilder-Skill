## PopupPanel

**Inherits:** [Popup](class_popup.md#class_Popup) **<** [Window](class_window.md#class_Window) **<** [Viewport](class_viewport.md#class_Viewport) **<** [Node](class_node.md#class_Node) **<** [Object](class_object.md#class_Object)

A popup with a panel background.

### Description

A popup with a configurable panel background. Any child controls added to this node will be stretched to fit the panel's size (similar to how [PanelContainer](class_panelcontainer.md#class_PanelContainer) works). If you are making windows, see [Window](class_window.md#class_Window).
