## NoiseTexture2D

**Inherits:** [Texture2D](class_texture2d.md#class_Texture2D) **<** [Texture](class_texture.md#class_Texture) **<** [Resource](class_resource.md#class_Resource) **<** [RefCounted](class_refcounted.md#class_RefCounted) **<** [Object](class_object.md#class_Object)

A 2D texture filled with noise generated by a [Noise](class_noise.md#class_Noise) object.

### Description

Uses the [FastNoiseLite](class_fastnoiselite.md#class_FastNoiseLite) library or other noise generators to fill the texture data of your desired size. **NoiseTexture2D** can also generate normal map textures.

The class uses [Thread](class_thread.md#class_Thread) s to generate the texture data internally, so [Texture2D.get_image()](class_texture2d.md#class_Texture2D_method_get_image) may return `null` if the generation process has not completed yet. In that case, you need to wait for the texture to be generated before accessing the image and the generated byte data:

::

    var texture = NoiseTexture2D.new()
    texture.noise = FastNoiseLite.new()
    await texture.changed
    var image = texture.get_image()
    var data = image.get_data()

<span id="class_NoiseTexture2D_property_seamless"></span>

[bool](class_bool.md#class_bool) **seamless** = `false` [ðŸ”—](class_noisetexture2d.md#class_NoiseTexture2D_property_seamless)

- void **set_seamless** ( value: [bool](class_bool.md#class_bool) )
- [bool](class_bool.md#class_bool) **get_seamless** ( )

If `true`, a seamless texture is requested from the [Noise](class_noise.md#class_Noise) resource.

 **Note:** Seamless noise textures may take longer to generate and/or can have a lower contrast compared to non-seamless noise depending on the used [Noise](class_noise.md#class_Noise) resource. This is because some implementations use higher dimensions for generating seamless noise.

 **Note:** The default [FastNoiseLite](class_fastnoiselite.md#class_FastNoiseLite) implementation uses the fallback path for seamless generation. If using a [width](class_noisetexture2d.md#class_NoiseTexture2D_property_width) or [height](class_noisetexture2d.md#class_NoiseTexture2D_property_height) lower than the default, you may need to increase [seamless_blend_skirt](class_noisetexture2d.md#class_NoiseTexture2D_property_seamless_blend_skirt) to make seamless blending more effective.

---

- [float](class_float.md#class_float) **get_seamless_blend_skirt** ( )
