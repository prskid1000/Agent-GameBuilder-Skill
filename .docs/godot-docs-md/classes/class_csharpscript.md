## CSharpScript

**Inherits:** [Script](class_script.md#class_Script) **<** [Resource](class_resource.md#class_Resource) **<** [RefCounted](class_refcounted.md#class_RefCounted) **<** [Object](class_object.md#class_Object)

A script implemented in the C# programming language, saved with the `.cs` extension (Mono-enabled builds only).

### Description

This class represents a C# script. It is the C# equivalent of the [GDScript](class_gdscript.md#class_GDScript) class and is only available in Mono-enabled Godot builds.
