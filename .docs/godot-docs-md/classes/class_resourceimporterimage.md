## ResourceImporterImage

**Inherits:** [ResourceImporter](class_resourceimporter.md#class_ResourceImporter) **<** [RefCounted](class_refcounted.md#class_RefCounted) **<** [Object](class_object.md#class_Object)

Imports an image for use in scripting, with no rendering capabilities.

### Description

This importer imports [Image](class_image.md#class_Image) resources, as opposed to [CompressedTexture2D](class_compressedtexture2d.md#class_CompressedTexture2D). If you need to render the image in 2D or 3D, use [ResourceImporterTexture](class_resourceimportertexture.md#class_ResourceImporterTexture) instead.
