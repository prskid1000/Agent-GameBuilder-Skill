## RenderSceneDataRD

**Inherits:** [RenderSceneData](class_renderscenedata.md#class_RenderSceneData) **<** [Object](class_object.md#class_Object)

Render scene data implementation for the RenderingDevice based renderers.

### Description

Object holds scene data related to rendering a single frame of a viewport. See also [RenderSceneData](class_renderscenedata.md#class_RenderSceneData), [RenderData](class_renderdata.md#class_RenderData), and [RenderDataRD](class_renderdatard.md#class_RenderDataRD).

 **Note:** This is an internal rendering server object. Do not instantiate this class from a script.
