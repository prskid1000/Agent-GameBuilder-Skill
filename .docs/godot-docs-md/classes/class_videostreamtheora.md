## VideoStreamTheora

**Inherits:** [VideoStream](class_videostream.md#class_VideoStream) **<** [Resource](class_resource.md#class_Resource) **<** [RefCounted](class_refcounted.md#class_RefCounted) **<** [Object](class_object.md#class_Object)

[VideoStream](class_videostream.md#class_VideoStream) resource for Ogg Theora videos.

### Description

[VideoStream](class_videostream.md#class_VideoStream) resource handling the Ogg Theora video format with `.ogv` extension. The Theora codec is decoded on the CPU.

 **Note:** While Ogg Theora videos can also have a `.ogg` extension, you will have to rename the extension to `.ogv` to use those videos within Godot.
