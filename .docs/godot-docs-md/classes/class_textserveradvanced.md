## TextServerAdvanced

**Inherits:** [TextServerExtension](class_textserverextension.md#class_TextServerExtension) **<** [TextServer](class_textserver.md#class_TextServer) **<** [RefCounted](class_refcounted.md#class_RefCounted) **<** [Object](class_object.md#class_Object)

An advanced text server with support for BiDi, complex text layout, and contextual OpenType features. Used in Godot by default.

### Description

An implementation of [TextServer](class_textserver.md#class_TextServer) that uses HarfBuzz, ICU and SIL Graphite to support BiDi, complex text layouts and contextual OpenType features. This is Godot's default primary [TextServer](class_textserver.md#class_TextServer) interface.
