## VisualShaderNodeTexture2DArrayParameter

**Inherits:** [VisualShaderNodeTextureParameter](class_visualshadernodetextureparameter.md#class_VisualShaderNodeTextureParameter) **<** [VisualShaderNodeParameter](class_visualshadernodeparameter.md#class_VisualShaderNodeParameter) **<** [VisualShaderNode](class_visualshadernode.md#class_VisualShaderNode) **<** [Resource](class_resource.md#class_Resource) **<** [RefCounted](class_refcounted.md#class_RefCounted) **<** [Object](class_object.md#class_Object)

A visual shader node for shader parameter (uniform) of type [Texture2DArray](class_texture2darray.md#class_Texture2DArray).

### Description

This parameter allows to provide a collection of textures for the shader. You can use [VisualShaderNodeTexture2DArray](class_visualshadernodetexture2darray.md#class_VisualShaderNodeTexture2DArray) to extract the textures from array.
