A [Cubemap](class_cubemap.md#class_Cubemap) parameter node to be used within the visual shader graph.

### Description

Translated to `uniform samplerCube` in the shader language. The output value can be used as port for [VisualShaderNodeCubemap](class_visualshadernodecubemap.md#class_VisualShaderNodeCubemap).
