**Inherited By:** [EditorExportPlatformLinuxBSD](class_editorexportplatformlinuxbsd.md#class_EditorExportPlatformLinuxBSD), [EditorExportPlatformWindows](class_editorexportplatformwindows.md#class_EditorExportPlatformWindows)

Base class for the desktop platform exporter (Windows and Linux/BSD).

### Description

The base class for the desktop platform exporters. These include Windows and Linux/BSD, but not macOS. See the classes inheriting from this one for more details.
