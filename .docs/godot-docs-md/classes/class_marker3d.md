## Marker3D

**Inherits:** [Node3D](class_node3d.md#class_Node3D) **<** [Node](class_node.md#class_Node) **<** [Object](class_object.md#class_Object)

Generic 3D position hint for editing.

### Description

Generic 3D position hint for editing. It's just like a plain [Node3D](class_node3d.md#class_Node3D), but it displays as a cross in the 3D editor at all times.
