## StyleBoxEmpty

**Inherits:** [StyleBox](class_stylebox.md#class_StyleBox) **<** [Resource](class_resource.md#class_Resource) **<** [RefCounted](class_refcounted.md#class_RefCounted) **<** [Object](class_object.md#class_Object)

An empty [StyleBox](class_stylebox.md#class_StyleBox) (does not display anything).

### Description

An empty [StyleBox](class_stylebox.md#class_StyleBox) that can be used to display nothing instead of the default style (e.g. it can "disable" `focus` styles).
