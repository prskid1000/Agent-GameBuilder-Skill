## Texture

**Inherits:** [Resource](class_resource.md#class_Resource) **<** [RefCounted](class_refcounted.md#class_RefCounted) **<** [Object](class_object.md#class_Object)

**Inherited By:** [Texture2D](class_texture2d.md#class_Texture2D), [Texture3D](class_texture3d.md#class_Texture3D), [TextureLayered](class_texturelayered.md#class_TextureLayered)

Base class for all texture types.

### Description

**Texture** is the base class for all texture types. Common texture types are [Texture2D](class_texture2d.md#class_Texture2D) and [ImageTexture](class_imagetexture.md#class_ImageTexture). See also [Image](class_image.md#class_Image).
